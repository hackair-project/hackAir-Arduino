SDS011

General Information

  • Type: Laser
  • Connection: Serial
  • Fan: Yes
  • Power saving: –
  • Library Reference: SENSOR_SDS011

Other sensors of the same family (SDSxxx) are potentially supported since most of them use the same data packet format.

How to connect

This sensors comes with a USB-to-Serial adapter. To connect the sensor to the Arduino we recommend cutting the wire that goes to the serial adapter and connecting the wires to the Arduino as follows:

Sensor Pin Description Arduino Pin WiFi Shield pin
GND Power Ground GND GND
VCC Power 5V 5V
RX Receive data 7 STX
TX Transmit data 8 SRX
