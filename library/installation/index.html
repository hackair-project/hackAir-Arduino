Arduino Library Installation

Installation

The latest version of the hackAIR Arduino library is always available for download at the project’s Github page. The easiest way to add the library to your Arduino IDE is by using the release .ZIP file. You can download this file from the “Releases” page on Github. The file you need is called hackAir-vX.Y.Z.zip, always prefer the latest version.

At the Arduino IDE, navigate to Sketch → Include Library → Add .ZIP library and select the file you just downloaded. The library and the example sketches should now be available.
